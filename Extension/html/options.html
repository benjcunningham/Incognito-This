    Frequently Asked Questions

    How do I switch multiple tabs at once?
    While holding the Control key (Windows/Linux) or Command key (Mac) select the tabs that you want to switch (or select a range of tabs by holding the Shift key while selecting tabs), then click the Incognito This! toolbar button. You can also use the Omnibox to switch the entire window (or all windows) with a single command (see next question).
    How do I use Incognito This! with Chrome's Omnibox (URL bar)?
    Select all the text currently inside the Omnibox, and delete it. Then type "incognito" (without the quotes) followed by the space bar or Tab key, and enter a command specifying what scope of tabs you would like to be switched (such as "tab" [currently active tab], "tabs" [all selected tabs within the window {see previous question}], "window" [the current window], or "all" [every tab open in Chrome]), a URL, or a Google search query, or enter "options" to open this Options page. You can also begin typing the name or URL of a bookmark or history item and it will appear in the list of options beneath the Omnibox. The first option in this list under "Run Incognito This! command:…" will be the one that Incognito This! will execute when the Enter key is pressed.
    How do I enable Incognito This! to run in Incognito mode for full functionality?
    Go to the Extension Manager, find the section for Incognito This!, and check the box next to "Allow in incognito". Checking the box next to "Allow access to file URLs" will also help Incognito This! function more well, but it is generally not needed in order to work smoothly.
    I want to use Incognito This!, but I don't want other people to know that I have it. What can I do to hide it?
    Right-click (Windows/Linux) or Control-click (Mac) the Incognito This! toolbar button and select "Hide Button" from the menu that appears. You can continue to use Incognito This! by entering commands via the Omnibox (see previous question) or enabling one of the options above to switch tabs with a key command or the context menu. If you want to unhide the button, go to the Extension Manager, find the section for Incognito This!, and click the "Show button" button within it.
    Does the "Save" button actually do anything?
    No. All of the preferences automatically save as soon as you change them. That button is just there to make you comfortable in case you want to feel sure that they saved. Seriously, though, that button does absolutely nothing.
